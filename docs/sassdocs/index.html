    align

    @mixin align($position: center) { ... }

    Description

    Applies flex alignment on elements

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $position

    Align direction ( top | middle | bottom | left | center | right )

    		Directionalcenter

    Example

    # use:

align();
align(top);
align(middle);
align(bottom);
align(left);
align(center);
align(right);
align(around);
align(between);
align(evenly);

# code:

.className {
    @include align(top);
}

# compiled:

.className {
    align-items: flex-start;
}

    lupe

    @mixin lupe($multiplier, $selectors) { ... }

    Description

    Multiply prop values by number

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $multiplier

    Number to multiply values

    		Number— none
    $selectors

    map with props and values

    		Map— none

    Example

    # code

$props: (
  font-size: 12px,
  padding-left: 10px
);

.className {
  @include lupe(0.75, $props);
}

# compiled

  font-size: 9px;
  padding-left: 7.5px;
}
